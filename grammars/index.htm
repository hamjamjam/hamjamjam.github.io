Basic Grammars 3D

Tools used: Rhino, Laser cutter

Fall 2015 - Collaborator: Renaud Danhaive

Basic grammars was about looking into applying shape grammar logic in 3D to generate a design space. In order to do this, we had to come up with a base shape which required iterating through a topology optimzation process. Ultimately, we chose a shape with a lot of rotational and reflective symmetry inspired by a japanese wood join.



Above: two shapes in considerations. Below: sample designs using two shapes above.



Initial ideas examined using two shapes, both interfacing with each other in the same way, but one with a 90 degree bend to allow a larger design space. We chose to use only the first shape because we felt that the addition of the 90 degree turn did not alter the topology of the design space significnatly enough to warrant the additional manufacturing difficulty.



The shape we chose has eight orders of symmetry and interfaces with itself from either end. We defined two spatial relations, shown below, which generate different designs based on labelling convention.



Labelling Labelling can be used to make a spatial relation, or rule, more rigid. For example, placing a label on the corner of one side of the base shape means that further base shapes can only be added in one place.

In the example below, labelling for both rules is used to generate the following design, named the fighter jet.



The rules define how the shape is built up, this labelling generates the following end result.



We iterated over possible shape labellings to generate 11 unique designs. Below is one additional design stemming from a new spatial relation. It turns out that our initial shape interface with itself at an angle as well.
