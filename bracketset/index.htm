Bracket set

Tools used: Rhino, Laser cutter

Spring 2014 - Collaborators: Tara Ebsworth & Kris Swick

The bracket set itself is the last piece that supports the roof of the building as described in the Ying-Zao Fa-Shi, a technical treatise on Chinese architecture with a very algorithmic approach. This was then extended to a full building slice.







Wider buildings call for larger bracket sets which become increasingly complicated as size increases. Above is the fully constructed set with the diagonal beam passing through representing how it connects to the sloping roof. Below is the slightly less complicated bracket set.



Some parts were 3D printer in order to experiment with how well the modular design coped with a smaller scale.





This was then extended to a building slice in order to demonstrate how multiple bracket sets would sit undereath the roof and hold it up.
